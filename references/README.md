# Reference Repositories

This folder is for shallow local clones of the two upstream projects used as architecture references:

- `career-ops`: https://github.com/santifer/career-ops
- `Resume-Matcher`: https://github.com/srbhr/Resume-Matcher

They are intentionally ignored by git because they are third-party source trees. Fetch them with:

```powershell
.\scripts\fetch_reference_repos.ps1
```
